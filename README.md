# kokoa-clone-2020

-----------------------------------------------------------------------------
## 프로젝트 설명  
노마드 코더의 카카오톡 클론 코딩 챌린지에 참여하여 카카오톡의   
실제 템플릿을 직접 따라 만들어보는 과정을 통해 기본적인 CSS 레이아웃 구성 및 animation, media query와 같은  
기능들을 실제로 사용해보며 HTML 및 CSS에 대한 이해를 높이는 프로젝트임
